Style Features
  • Custom navigation blocks

    Navigation blocks

    This style has several customizable navigation blocks:

    - Header navigation (this menu)
    - Left side bar.
    - Right side bar (not shown by default).

    Header navigation

    Header navigation can have several types of items:

    - Simple links (see "Hide Menu" on right)
    - Popups with content (this menu)
    - Popups with list of links (parent menu)
    - Popups with list of links can have children popup menus (like this)

    Side bars

    Left and right sidebars can have any content. You can put ads, links,
    forms there. You can separate content into sections (see "Search" or
    "Login" sections on left).

  • Custom colors

    Color variations

    Style comes with only 7 predefined color variations that
    you can preview by clicking color buttons on the right,
    but you can create many more color variations yourself.

    Editing colors

    To create your own color variation to match your website,
    use ColorizeIt online style editor. Follow this link.

    ColorizeIt editor allows you to change color's hue
    component to whatever you want and you can also change
    lightness and saturation to some extent.

    Unlike color pickers that you see in WordPress themes,
    ColorizeIt editor changes not only CSS files, but also
    images. Downside is you cannot select specific link or
    part of forum - editor changes all colors with same hue.
    To change a specific link you'll need to edit CSS files.

    Tutorial

    Tutorial is available in documentation.

  • Compatible with prosilver

    Compatibility

    This style uses prosilver (default phpBB 3 style) template files,
    which means it is compatible with most phpBB 3 mods.

    Note: installing mods might require some tweaking because theme
    (css) files are different from prosilver style.

  • Easy to integrate

    Integration

    This style is written in HTML5 and uses jQuery, making it easier to
    add your favorite slideshows and custom menus.

    All CSS selectors start with ".phpbb", removing potential CSS conflicts
    with custom menus or wrappers, making it easier to integrate forum into
    existing website layout.

  • More features:

    Other features

    This style has few nice features that most other phpBB styles do not have.
    Most of these features have big impact on usability, making forum much easier
    to use.

    Collapsable categories

    You can show/hide categories on forum index, browser will remember your
    selection and style will hide selected categories on every page load.

    New private messages notification

    This style has a custom block that notifies you on new private message.
    phpBB uses popup window to do that, but these days most users have popup
    blockers and might miss new private message. This style will show a big
    warning below top navigation informing you about new or unread private message(s).

    Profile on left side

    Poster's profile is shown on the left side, like in other forum systems.
    Each poster has default avatar that you can customize and poster's links
    (such as home page, icq, msn) are shown in popup.

    Highlighting new posts

    Forums with new posts not only have custom image, but are also highlighted
    with a different color, making it easier to see where new posts are.
